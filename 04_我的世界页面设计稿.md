# 04｜“我的世界”页面设计稿

> 页面：`/world`  
> 目标：把家乡、学校、旅行、游戏、特点与生活兴趣做成一个沉浸式个人世界地图。

---

## 1. 页面定位

`/world` 是本次改版最重要的个性化页面。

它不是普通“兴趣爱好”页面，而是一个把你个人经历和兴趣连接起来的世界地图。

核心概念：

```text
我的世界地图
从家乡出发，在学校成长，去旅行探索，也在游戏里进入另一个宇宙。
```

标题：

```text
LQPP World Map
我的世界
```

副标题：

```text
这里收藏我的家乡、学校、旅行、游戏和一些属于我的特点。
```

---

## 2. 页面结构

```text
Hero：我的世界地图入口
快速导航：家乡 / 学校 / 旅行 / 游戏 / 我的特点
Section 1：出发的地方：家乡
Section 2：成长坐标：学校
Section 3：探索轨迹：旅行
Section 4：游戏宇宙：我的游戏
Section 5：我的特点：关键词卡片
Section 6：世界更新日志
```

---

## 3. Hero 设计

文案：

```text
欢迎来到我的世界地图。
每个人都有自己的坐标。
我的坐标由家乡、学校、旅行、游戏和无数个普通但珍贵的瞬间组成。
```

按钮：

```text
看看我的家乡
查看旅行轨迹
进入游戏宇宙
```

视觉：

- 背景是抽象地图 + 星空轨道。
- 中间可以有一个星球或地图节点图形。
- 节点围绕：家乡、学校、旅行、游戏、特点。
- 滚动时节点可以轻微漂浮，但不要影响阅读。

---

## 4. 快速导航

锚点导航：

```text
出发的地方｜成长坐标｜探索轨迹｜游戏宇宙｜我的特点
```

要求：

- 点击平滑滚动到对应区块。
- 当前区块高亮。
- 移动端横向滚动。

---

## 5. 家乡：出发的地方

标题：

```text
出发的地方
```

副标题：

```text
家乡是我世界地图上的第一个坐标。
```

内容建议：

```text
家乡名称
家乡关键词：山 / 海 / 河流 / 小城 / 美食 / 方言 / 天气 / 记忆
我对家乡的印象
代表性照片 3-6 张
家乡美食 / 地标 / 童年记忆
```

卡片结构：

```text
左侧：家乡照片或地图插画
右侧：文字介绍
底部：关键词标签
按钮：查看家乡相册 / 阅读家乡文章
```

示例文案：

```text
这里是我出发的地方。它可能不是地图上最显眼的城市，但它保存着我的很多记忆：熟悉的街道、吃过的味道、见过很多次的天空，还有每次离开后仍然会想起的生活感。
```

---

## 6. 学校：成长坐标

标题：

```text
成长坐标
```

副标题：

```text
学校记录了我学习、变化和慢慢确定方向的过程。
```

字段：

```text
学校名称
时间段
专业 / 年级 / 阶段
喜欢的课程
重要经历
校园照片
学到的东西
```

展示方式：

- 时间线。
- 校园照片卡片。
- 课程标签。
- 当前学校或当前阶段高亮。

示例：

```text
202X - 至今
学校名称
关键词：编程 / 课程 / 社团 / 比赛 / 朋友 / 成长
描述：这一阶段让我开始认真学习技术，也让我想把自己的想法做成真实可访问的网站。
```

---

## 7. 旅行：探索轨迹

标题：

```text
探索轨迹
```

副标题：

```text
每一次出发，都会在我的地图上点亮一个新的坐标。
```

字段：

```text
城市 / 地点
时间
关键词：风景 / 美食 / 朋友 / 独自 / 计划外
代表照片
一句话记忆
相关文章或相册链接
```

设计：

- 第一版可以做“旅行卡片 + 时间线”。
- 第二版可以做中国地图或世界地图，点亮去过的城市。
- 点击城市显示照片和故事。

旅行卡片文案示例：

```text
这次旅行不一定走了很远，但我记住了那天的风、路边的店、拍下照片时的心情。旅行对我来说不只是到达一个地点，更像是给自己的生活增加一个新坐标。
```

---

## 8. 游戏：游戏宇宙

标题：

```text
游戏宇宙
```

副标题：

```text
游戏也是一种探索世界的方式。
```

字段：

```text
游戏名称
游戏类型
状态：正在玩 / 已通关 / 偶尔上线 / 想玩
喜欢原因
截图或封面
我的记忆点
```

分类：

```text
正在玩
玩过并喜欢
想玩
和朋友一起玩的游戏
```

如果你喜欢 Minecraft，可以单独加：

```text
我的 Minecraft 世界
- 建筑截图
- 生存记录
- 服务器经历
- 红石 / 建筑 / 探险标签
```

---

## 9. 我的特点

标题：

```text
构成我的一些碎片
```

卡片：

```text
喜欢记录
喜欢折腾网站
喜欢探索新地方
喜欢游戏里的自由感
喜欢用技术做自己的东西
喜欢把记忆整理成可浏览的页面
```

每张卡片：

```text
icon
关键词
一句说明
相关链接：文章 / 相册 / 世界节点
```

---

## 10. 世界更新日志

标题：

```text
我的世界更新日志
```

示例：

```text
2026.04：开始重构个人博客
2026.04：加入 3D 星空相册
2026.05：计划加入我的世界页面
未来：加入旅行地图、留言板、项目页
```

这个模块能让网站看起来像持续生长的个人项目。

---

## 11. 给 vibe coding 的提示词

```text
请新增 /world 页面，命名为“我的世界”或“LQPP World Map”。页面要像一张个人世界地图，包含家乡、学校、旅行探索、游戏宇宙和我的特点五个核心区块。顶部有沉浸式 Hero 和锚点导航。每个区块使用卡片、时间线、照片和标签展示。整体风格延续星空、地图坐标、数字花园和轻量 3D 感。移动端要保持单列清晰阅读，桌面端可以使用地图节点、卡片网格和时间线组合。
```
