# 02｜首页改版设计稿

> 页面：`/`  
> 目标：让访客在 10 秒内知道你是谁、网站有什么、为什么值得继续看。

---

## 1. 首页定位

首页不只是博客列表，而是进入你个人世界的入口。

推荐首屏文案：

```text
你好，我是 LQPP。
欢迎来到我的个人世界。
这里记录我的生活、技术、家乡、学校、旅行、游戏，以及一些慢慢长大的想法。
```

副文案：

```text
我把每一篇文章、每一张照片、每一次探索，都放进这个属于自己的数字星球。
```

按钮：

```text
进入我的世界
阅读文章
打开记忆相册
```

---

## 2. 首页模块结构

### Section 1：Hero 首屏

布局：左侧文案，右侧个人卡片。

左侧内容：

```text
小标签：Welcome to LQPP World
主标题：在代码、生活和记忆里，搭建自己的小宇宙
说明：记录生活、技术、旅行、游戏和成长
按钮：我的世界 / 最新文章 / 联系我
```

右侧个人卡：

```text
头像
昵称：LQPP
身份标签：学生 / 博客作者 / 代码学习者 / 旅行探索者 / 游戏玩家
当前状态：正在建设自己的数字花园
联系方式：Email / GitHub / 微博 / B站
```

视觉：

- 背景使用深色星空渐变或浅色柔和渐变。
- 头像外圈有渐变发光边框。
- 个人卡使用毛玻璃效果。
- 首屏不要堆太多文字，要留白。

---

### Section 2：四个入口卡片

标题：

```text
我的世界由这些部分组成
```

| 卡片 | 描述 | 跳转 |
|---|---|---|
| 我的世界 | 家乡、学校、旅行、游戏和我的特点 | `/world` |
| 思考碎片 | 技术笔记、生活随笔、灵感记录 | `/articles` |
| 记忆星图 | 照片、相册、3D 星空照片墙 | `/photos` |
| 我的档案 | 教育经历、技术栈、联系方式 | `/about` |

交互：

- hover 时上浮 4px。
- 背景出现柔和光斑。
- 每张卡片使用不同 icon。
- 移动端改为单列。

---

### Section 3：当前状态

增强个人感的模块：

```text
当前状态
- 正在学习：前端开发 / 后端基础 / AI 工具使用
- 最近在玩：Minecraft / Steam 游戏 / 其他游戏
- 最近想去：某城市 / 某景点
- 正在完善：我的个人博客 v2.0
```

也可以做成横向滚动：

```text
Now Learning: Web Development · Now Playing: Minecraft · Now Building: LQPP World
```

---

### Section 4：最新文章

标题：

```text
最近写下的东西
```

副标题：

```text
一些技术笔记、生活片段和突然出现的想法。
```

文章卡字段：

```text
分类标签
标题
摘要
日期
浏览量
阅读全文按钮
```

约束：

- 首页最多展示 3 篇文章。
- 文章必须有摘要；没有摘要时自动截取正文前 80 字。
- 上线前清理测试标题和无意义正文。

---

### Section 5：精选相册 / 记忆星图

标题：

```text
记忆星图
```

文案：

```text
把生活片段放进一个可以浏览的星空里。
```

内容：

- 左侧展示 3-6 张精选照片。
- 右侧说明普通相册和 3D 星空相册。
- 按钮：`打开 3D 星空相册`、`查看全部照片`。

---

### Section 6：技术栈预览

标题：

```text
我的工具箱
```

示例：

```text
Frontend：HTML / CSS / JavaScript / React / Next.js / Tailwind CSS
Backend：Node.js / Python / 数据库
Tools：Git / GitHub / Markdown / AI Coding
Learning：算法 / 全栈开发 / Web 动效 / 3D 交互
```

约束：

- 不要写完全不会的技术。
- 可以分为“正在使用”“正在学习”“想探索”。

---

推荐文案：

快捷入口：首页 / 文章 / 相册 / 我的世界 / 关于我
联系我：Email / GitHub / 微博 / QQ / 微信二维码

```

## 3. 首页动效

允许：

- 首屏淡入上移。
- 星点背景缓慢漂浮。
- 卡片 hover 上浮。
- 按钮 hover 出现柔和光晕。
- 页面切换淡入。

避免：

- 大量旋转。
- 频繁闪烁。
- 页面滚动劫持。
- 影响阅读的背景动画。

---

## 4. 首页移动端顺序

最新文章

```
