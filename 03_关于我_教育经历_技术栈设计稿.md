# 03｜关于我、教育经历与技术栈设计稿

> 页面：`/about`  
> 目标：展示头像、联系方式、个人介绍、教育经历、技术栈、特点与成长方向。

---

## 1. 页面定位

`/about` 不是普通简历页，而是“个人档案页”。

它要回答：

1. 你是谁？
2. 你在学习什么？
3. 你的教育经历是什么？
4. 你会什么技术？
5. 如何联系你？

推荐标题：

```text
LQPP Profile
我的档案
```

副标题：

```text
一个正在用代码、文字和照片搭建自己世界的人。
```

---

## 2. 页面布局

桌面端：

```text
左侧 30%：sticky 个人信息卡
右侧 70%：个人介绍、教育经历、技术栈、特点、联系
```

移动端：

```text
个人信息卡
我是谁
教育经历
技术栈
个人特点
联系方式
```

---

## 3. 左侧个人信息卡

内容：

```text
头像
昵称：LQPP
一句话介绍：正在搭建自己的数字花园
身份标签：学生 / 开发学习者 / 博客作者 / 旅行探索者 / 游戏玩家
所在地：可写家乡或当前城市，不想暴露可写“中国 · 地球在线”
联系方式：Email / GitHub / 微博 / B站 / QQ / 微信二维码
按钮：联系我 / 查看我的世界
```

头像要求：

- 圆形或圆角方形。
- 外圈渐变描边。
- hover 时轻微放大。
- 缺失头像时使用 `LQPP` 字母头像。

隐私要求：

- Email 可以展示。
- 微信、QQ 建议二维码弹窗，不要直接公开太多信息。
- 所有外链新窗口打开。
- 邮箱使用 `mailto:`。

---

## 4. “我是谁”模块

标题：

```text
我是谁
```

示例文案：

```text
你好，我是 LQPP。
我喜欢记录生活，也喜欢研究技术。对我来说，网站不只是一个展示页面，而是一个可以慢慢成长的个人世界。

我会把学习过程、旅行照片、家乡记忆、学校经历、游戏片段和一些突然出现的想法放在这里。希望这个网站能像一本不断更新的个人地图，记录我从哪里来、正在做什么、想去哪里。
```

可替换关键词：

```text
好奇 / 慢热 / 喜欢探索 / 喜欢游戏 / 喜欢拍照 / 喜欢折腾网站 / 正在学习编程
```

---

## 5. 教育经历模块

标题：

```text
教育经历
```

建议使用时间线组件。

字段：

```text
时间段：202X - 202X
学校：学校名称
阶段：高中 / 大学 / 专业 / 班级
关键词：课程、社团、比赛、项目、成长经历
描述：这一阶段对我的影响
```

示例：

```text
202X - 至今
学校名称 / 专业方向
在这里我开始系统学习技术，也逐渐意识到自己想把兴趣、记录和作品结合起来。这个网站就是其中一个长期项目。
```

视觉：

- 左侧发光圆点。
- 中间连线。
- 当前阶段高亮。
- 每段经历使用卡片展示。
- 可以加入校园照片或课程标签。

