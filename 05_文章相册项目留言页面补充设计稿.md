# 05｜文章、相册、项目、留言等页面补充设计稿

> 目标：在现有文章和相册基础上，补充更多能体现个人特色的页面与模块。

---

## 1. 文章页 `/articles`

当前文章页已有文章目录、搜索和文章列表。建议加强分类和摘要。

页面标题：

```text
思考碎片
```

副标题：

```text
这里放着我的技术笔记、生活随笔、旅行记录和一些突然出现的想法。
```

分类建议：

```text
全部
技术笔记
生活记录
旅行探索
游戏随笔
学校日常
家乡记忆
网站更新
```

文章卡字段：

```text
标题
摘要
分类标签
发布日期
浏览量
封面图，可选
阅读时间
```

约束：

- 所有文章尽量有摘要。
- 没有摘要时自动截取正文前 80 字。
- 上线前清理测试文章。
- 文章列表支持搜索和分类筛选。

---

## 2. 文章详情页 `/posts/[slug]`

推荐结构：

```text
返回文章列表
文章头部：标题、日期、分类、浏览量、阅读时间
正文：Markdown 内容
桌面端右侧：目录 TOC
底部：上一篇 / 下一篇 / 相关推荐
评论或留言入口
```

视觉：

- 正文宽度建议 720px - 860px。
- 图片统一圆角。
- 代码块带复制按钮。
- 引用块带左侧高亮线。
- 文章结尾可以放作者卡片。

---

## 3. 相册页 `/photos`

当前相册页已经有普通相册和 3D 星空相册。建议包装为“记忆星图”。

页面标题：

```text
记忆星图
```

副标题：

```text
把生活片段放进一个可以浏览的星空里。
```

分类建议：

```text
全部
家乡
学校
旅行
生活
游戏
随手拍
```

入口：

```text
普通相册：适合快速浏览
3D 星空相册：适合沉浸浏览
```

照片字段：

```text
照片
标题
拍摄时间
地点
分类
一句话记忆
```

约束：

- 图片懒加载。
- 点击图片打开大图预览。
- 移动端避免复杂 3D 操作影响性能。
- 3D 星空相册入口在手机端提供普通模式备选。

---

## 4. 项目 / 技术作品模块

可以先放在 `/about`，以后再独立成 `/projects`。

标题：

```text
我正在做的东西
```

项目卡字段：

```text
项目名称
项目描述
技术栈
项目截图
GitHub 链接
在线预览
当前状态：进行中 / 已完成 / 想重构
```

第一个项目建议就是本站：

```text
LQPP World
一个记录生活、技术、旅行、游戏和成长的个人网站。它既是博客，也是我的数字花园。
```

## 5. 留言 / 联系页 `/guestbook` 或 `/contact`

页面标题：

```text
给我留句话
```

副标题：

```text
无论是技术、博客、旅行、游戏，还是一句简单的你好，都欢迎留下。
```

模块：

```text
联系卡片
留言表单
友链区域
社交平台入口
```

留言表单字段：

```text
昵称
邮箱，可选
网站，可选
留言内容
提交按钮
```

约束：

- 留言需要防垃圾评论。
- 后台审核后展示。
- 不展示用户邮箱。
- 表单提交后显示成功状态。

---

## 6. 时间线页面 `/timeline`，可选

如果你想让网站更像成长记录，可以加时间线。

内容来源：

```text
网站更新
旅行节点
学校阶段
技术学习节点
重要照片
游戏记录
```

如果不想增加导航，可以把时间线放在“我的世界”页面底部。
